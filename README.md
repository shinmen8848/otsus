## üöÄ Getting Started - Relive Our Journey

To run this project locally, you'll need to have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/tomoe-nanami.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Set up your environment variables:**
    Create a `.env` file and add your Supabase credentials.
    ```
    VITE_SUPABASE_URL=your-supabase-url
    VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
